Java 17 以上で実行できます。ブラウザ上のカード操作は JavaScript で動かし、Java はローカル配信用のサーバーです。

```powershell
cd C:\Users\masah\Documents\Codex\2026-07-18\yuni\outputs\union-arena-practice
javac UnionArenaServer.java

```

表示されたら、ブラウザで `http://localhost:8080` を開きます。終了はターミナルで `Ctrl+C` です。
